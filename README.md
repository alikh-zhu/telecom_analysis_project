Анализ оттока клиентов для телеком-компании

Цель проекта:
Ответить на вопрос: «Какие факторы влияют на отток клиентов?»

Стек технологий:
SQL, Python (Pandas), Power BI

Этапы работы

Загрузка и очистка данных (Python)

Импорт CSV-файла с данными клиентов.

Обработка пропусков и приведение типов данных к корректным.

Создание новых признаков, например, вычисление процента оттока.

Анализ средствами SQL

Подсчет оттока по разным категориям: пол, возраст, тип контракта, услуги.

Определение ключевых факторов, влияющих на уход клиентов.

Визуализация в Power BI

Дашборд с карточками и диаграммами:

Общее количество клиентов

Процент оттока (Churn Rate)

Отток по типу подписки и услугам

Динамика оттока по месяцам

Ключевые выводы

Клиенты с коротким сроком tenure уходят чаще.

Наличие дополнительных услуг (StreamingTV, OnlineSecurity) влияет на удержание.

Тип контракта (месячный vs долгосрочный) сильно коррелирует с оттоком.

Безбумажный биллинг (PaperlessBilling) и способ оплаты также имеют значение.
